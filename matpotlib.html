Plot#

La función básica y por defecto en matplotlib es plt.plot() que traza X frente a Y con lineas o marcadores. La sintaxis es la siguiente:

horas_estudio = horas_estudio.sort_values(by=['Scores'])

x = horas_estudio['Scores']
y = horas_estudio['Hours']
plt.xlabel('Notas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la x
plt.ylabel('Horas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la y
plt.plot(x, y, linestyle= 'solid')
plt.legend(['Línea relación horas estudio y notas'])
plt.show()

Aunque la sintaxis es muy simple, tiene varias interfaces y se puede crear de diferentes maneras.

plt.plot(x, y, 'bo')  # plot x e y usando circulos azules

[<matplotlib.lines.Line2D at 0x7f30d06b4b20>]

plt.plot(y, 'r+')     # y con puntos rojos

[<matplotlib.lines.Line2D at 0x7f30d0541420>]

Con la funcion subplot podemos dibujar varias gráficas sobre la misma ventana estableciendo varios parámetros:

fig, axs = plt.subplots(2, 2, figsize=(10,8))
axs[0, 0].plot(x, y)
axs[0, 0].set_title('Axis [0, 0]')
axs[0, 1].plot(x, y, 'bo')
axs[0, 1].set_title('Axis [0, 1]')
axs[1, 0].plot(x, y, 'r+')
axs[1, 0].set_title('Axis [1, 0]')
axs[1, 1].plot(x, y, 'd')
axs[1, 1].set_title('Axis [1, 1]')


#Podemos modificar las etiquetas independientemente por graficos, en el caso del objeto axes con la funcion .set_xlabels()
axs[0, 0].set_xlabel('Notas')
axs[0, 0].set_ylabel('Horas')

axs[0, 1].set_xlabel('Notas')
axs[0, 1].set_ylabel('Horas')

axs[1, 0].set_xlabel('Notas')
axs[1, 0].set_ylabel('Horas')

axs[1, 1].set_xlabel('Notas')
axs[1, 1].set_ylabel('Horas')


#Con el comando tight_layout() mantenemos automaticamente el equispaciado entre las gráficas
fig.tight_layout()

Fill Between#

Con el se puede rellenar el area comprendida entre dos funciones.

#Por ejemplo, generemos dos nuevas funciones que sean Y original +-5
y2=y-5
y3=y+5

plt.xlabel('Notas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la x
plt.ylabel('Horas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la y
plt.plot(x, y, linestyle= 'solid')

plt.fill_between(x,y2,y3, alpha=.5)
plt.xlabel('Notas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la x
plt.ylabel('Horas', fontdict={'fontsize':15}) # Ponemos una etiqueta a la y

Text(0, 0.5, 'Horas')
